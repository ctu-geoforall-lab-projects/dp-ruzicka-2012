Description
-----------
processingmanager 	- main plugin for QGIS Processing Framework
			- included Processing Manager and Workflow Builder
saga 			- SAGA Plugin which register its modules into QGIS Processing Framework
			- they can be reached through PRocessing Manager			
workflow_builder	- plugin that registered modules, that are saved as .xml file in ~/.qgis/python/workflows
inputparameters		- plugin that registers modules, that can load raster or vector data into QGIS
			- useful when working with Workflow Builder 

Install
-------
Add this directories into ~/.qgis/python/plugins/ directory
